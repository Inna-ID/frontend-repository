Фиксы CSS

Tagged in CSS 3 • 19 Апр, 2018

В ходе верстки нередко встречаются различные баги. Иногда это зависит от браузера или его версии. Здесь буду собирать баги, с которыми мне довелось столкнуться и приводить примеры их решения.

Дергание фиксированного блока при прокрутке колесом мышки, либо стрелками.

Допустим есть фиксированный header. Это похоже на плавающий баг, ибо иногда при прокрутке страницы хедер начинает колбасить

Решить проблему в некотором случае можно добавив блоку transform: translateZ(0);

			

	css
	
	//efgeefef
	header {
	   -webkit-transform: translateZ(0);
	   transform: translateZ(0);
	}
